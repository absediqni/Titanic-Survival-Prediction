# Titanic-Survival-Prediction
End-to-end machine learning project predicting Titanic passenger survival using structured data. Includes data cleaning, feature engineering, exploratory data analysis, and classification modeling.

## Project structure

- `data/` — raw Titanic dataset files (`train.csv`, `test.csv`)
- `notebooks/` — Jupyter notebooks for analysis and modeling
- `dashboards/` — dashboard assets and visualizations
- `requirements.txt` — Python dependencies

## Notebook

Open `notebooks/titanic.ipynb` to run the analysis step-by-step.
